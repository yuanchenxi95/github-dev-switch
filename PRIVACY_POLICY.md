# Privacy Policy for Github Dev Switch

**Last updated:** February 17, 2026

## Overview

Github Dev Switch is a Chrome extension that switches between github.com and github.dev. This extension does not collect, store, transmit, or share any user data.

## Data Collection

This extension does **not** collect any data of any kind. This includes but is not limited to:

- Personally identifiable information
- Health information
- Financial and payment information
- Authentication information
- Personal communications
- Location data
- Web history or user activity
- Website content

## Permissions

This extension requires the following browser permissions:

- **tabs** - Used solely to read and modify the current tab's URL to switch between github.com and github.dev.
- **declarativeContent** - Used solely to activate the extension icon when the user is on a supported GitHub page.

These permissions are used exclusively for the core functionality of the extension. No data accessed through these permissions is collected, stored, or transmitted.

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Changes to This Policy

If this privacy policy changes, the updated policy will be posted in this repository.

## Contact

If you have questions about this privacy policy, please open an issue in the [GitHub repository](https://github.com/nickyui99/github-dev-switch).
